Intermediate electives

Intermediate elective options 1 and 2 involve the exact same steps. However, they are focused on different data sets.

Option 1 covers 1) Venoco Bridge water surface elevation, 2) birds, and 3) water quality parameters.

Option 2 covers 1) vernal pool vegetation and 2) aquatic invertebrates.

Choose the option you want to complete based on the data set you are most interested in.

Set up

On GitHub:

  1. Create a new repository called intermediate-elective. Be sure to initialize with a README.
  2. Clone that repository to your computer (or virtual machine, if you are using that option). Make sure it is in the git or github folder.

In the Finder or File Explorer or RStudio:

  1. Create new folders for code and data in your repo.
  2. Add any data files you need to the data folder.
Actions to get this done

You will need to copy data files you need into your data folder.

In RStudio:

  1. Create a new .qmd in the code folder. Be sure to:
    • select the PDF option
    • uncheck the box to use the visual editor (so that you are using the source editor)
  2. Insert a code chunk to read in your packages.
  3. Insert a code chunk to read in your data.
  4. Render to .pdf to make sure that you can render.
  5. Stage, commit, and push your changes.
  6. Find your rendered PDF on your GitHub remote. Copy the link.
  7. Back in RStudio, edit your README to include a link to your rendered assignment PDF.
  8. Stage, commit, and push your changes.
  9. Start your elective!

Elective

What you will do in this elective

In class and in assignments, you have created figures to explore patterns in data. For your intermediate elective, you will create a figure that tells a story through more advanced customization.

Background

Tidy Tuesday is a project organized by data scientists to encourage open exploration of publicly available datasets.

Participants in Tidy Tuesday create a visualization from a new dataset that’s shared each week. Some examples include Bird Sightings at Sea, Pokemon, and Objects Launched into Space.

For your elective, you will choose one of these visualizations as inspiration to create a figure using your selected dataset.

Open and reproducible work

All the code to create the above visualizations is openly available. You can and should repurpose the code to make the inspiration visualizations for your own figure.

To get another person’s code onto your computer, you can:

  1. fork their repository (generally good but not a great option if there are lots of folders/files in their repo and you only want one or two)
  2. download the relevant folder to your computer
  3. copy paste their code

Objective

You should be able to create a visualization that closely matches your inspiration in aesthetics with different data and variables.

This means that you should be able to recreate the core visual components (e.g. geometries, plot types) while replacing the information/variables to fit your own dataset (e.g. axes, explanatory text).

This also means that you may have to install new packages, modify your inspiration’s code, and/or clean/wrangle your dataset to fit the visualization.

Note: if you are unsure which components of your inspiration you should be able to recrate, ask for help.

# Elective
