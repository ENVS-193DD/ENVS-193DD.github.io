Wrangling the data

Pseudocode

Write pseudocode to create a data frame called birds_wy2025 that:

  • only includes the 2025 water year,
  • excludes repeat observations, and
  • includes observations from the following eBird groups: Waterfowl; New World Sparrows; Herons, Ibis, and Allies; Shorebirds

Pseudocode:

  • create object called birds_wy2025 from birds
  • filter date to keep observations after Sept 30 2024 and before Oct 1 2025
  • filter repeat_observations to only include “No” rows
  • filter ebird_group to only include Waterfowl; New World Sparrows; Herons, Ibis, and Allies; Shorebirds

Running code

We’ll add a new function at the end of this code to insert the lowest taxonomic group for each eBird group name.

We’ll use mutate() (a function you already know) along with case_when(), which is a new function, to replace values in a column.

# creating new object from birds
birds_wy2025 <- birds |> 
  # filter to include:
  # observation_dates in the 2025 water year
  # observations that are not repeats
  # eBird groups in the 4 groups of interest
  filter(observation_date > "2024-09-30" & observation_date < "2025-10-01",
         repeat_observation == "No",
         e_bird_group %in% c("Waterfowl",
                             "New World Sparrows",
                             "Herons, Ibis, and Allies",
                             "Shorebirds")) |> 
  # create a new column called lowest group
  mutate(lowest_group = case_when(
    # lowest group for sparrows is family
    e_bird_group == "New World Sparrows" ~ "Passerellidae",
    # lowest group for herons is family
    e_bird_group == "Herons, Ibis, and Allies" ~ "Ardeidae",
    # lowest group for shorebirds is order
    e_bird_group == "Shorebirds" ~ "Charadriiformes",
    # lowest group for waterfowl is family
    e_bird_group == "Waterfowl" ~ "Anatidae"
  ))

Figure out the species in each group.

You can combine wrangling functions to get subsets of information from a dataset.

At the end of this chunk of code, you should end up with a dataframe with the all the species in all eBird groups.

# create a new object called group_species from birds_wy2025
group_species <- birds_wy2025 |>

  group_by(e_bird_group) |> 
  # find the unique species in each group
  distinct(species) |>

herons_total_count <- birds_wy2025 |> 
  # filter to only include wading birds

Seasonal differences in species richness

Guiding question

In what season is species richness highest for each eBird group?

What is species richness?

Species richness is the number of species that occur in a given sampling unit. In this case, we are interested in the number of species in a given group in a given season in the 2025 water year. Thus, we are calculating species richness for four different seasons for four different eBird groups.

Pseudocode for summarizing

When used with group_by() and summarize(), n_distinct() allows you to count the number of distinct values in a group or multiple groups.

We can use this combination of functions to determine the number of species observed in a given eBird group in a given season.

Write pseudocode to calculate the number of species observed in each eBird group in a given season.

Pseudocode:

  • create new object from birds_wy2025
  • “roll up” Belding’s subspecies to be considered with Savannah Sparrows
  • group by ebird group and season
  • count number of species in each group in each season
  • ungroup dataframe
What is roll up?

Belding’s is a subspecies of Savannah Sparrow. When doing any calculations of species richness, it wouldn’t make sense to include both Savannah Sparrow and Belding’s Savannah Sparrow.

Instead of counting subspecies as species, researchers tend to “roll up” subspecies; that is, count subspecies as the higher species. In this example, we’ll use mutate() and case_when() to roll up the subspecies to make sure that Belding’s (the subspecies) is counted as Savannah (the species).

Running code

# create new object from birds_wy2025
season_spp_count <- birds_wy2025 |>
  # roll up subspecies
  mutate(species = case_when(
    # whenever Belding's appears, replace with Savannah Sparrow
    species == "Savannah Sparrow (Belding's)" ~ "Savannah Sparrow",
    # keep all other values in species
    .default = species
  )) |> 
  # group data frame by eBird group and season
  group_by(e_bird_group, season) |> 
  # count unique species that appear in each season in each group
  summarize(num_spp = n_distinct(species)) |> 
  # ungroup data frame
  ungroup() |> 
  # relevel season to be in correct order for water year
  mutate(season = fct_relevel(season,
                              "Fall", "Winter", "Spring", "Summer"))

Visualizing

# base layer: ggplot
ggplot(data = season_spp_count,
       # x-axis should be season, y-axis should be number of species
       mapping = aes(x = season,
                     y = num_spp,
                     fill = season)) +
  # first layer: columns for species richness
  geom_col() +
  # custom color for seasons
  scale_fill_manual(values = c(
    "Fall" = "#287984",
    "Winter" = "#9c8020",
    "Spring" = "#589101",
    "Summer" = "#4987c9"
  )) +
  # get rid of space between columns and x-axis
  scale_y_continuous(expand = c(0, 0),
                     limits = c(0, 12)) +
  # different theme from default
  theme_bw() +
  # facet figure to have one panel for each eBird group
  facet_wrap(~ e_bird_group) +
  # relabel x-and y-axes
  labs(x = "Season",
       y = "Species richness") +
  # take out the legend
  theme(legend.position = "none")

Interpretation

When is species richness highest and lowest for each group?

Species richness for Herons is highest in Spring and Summer, but lowest in Fall and Winter (low throughout though).

Species richness for New World Sparrows is highest in Winter and lowest in summer.

Shorebird species richness is highest in winter and spring, but lowest in fall and summer.

Waterfowl species richness is highest in winter and spring and lowest in fall and summer.

Seasonal differences in total counts

Guiding question

What are differences in total counts between seasons?

How does this differ (or not) from differences in species richness between seasons?

What are total counts?

In each survey, some number of individuals are counted for a species.

For example, you might see a flock of 12 Song Sparrows at the beginning of your survey, and another flock of 5 Song Sparrows at the end.

For that survey, your total count of Song Sparrows is 17. However, there is no way to know for sure if the 5 sparrows from the end of your survey are different from the 12 you saw at the beginning.

For this reason, we can only calculate “total counts”: the total number of counted individuals in a given survey.

Running code to summarize data

First, write code to calculate the number of individuals counted in each eBird group in each season.

# create new object from birds_wy2025
season_bird_count <- birds_wy2025 |>
  # group data frame by eBird group and season
  group_by(e_bird_group, season) |> 
  # calculate total counts for each group in each season
  summarize(total_count = sum(count)) |> 
  # ungroup data frame
  ungroup() |> 
  # relevel seasons 
  mutate(season = fct_relevel(season,
                              "Fall", "Winter", "Spring", "Summer"))

Guiding question

How do total counts change through time for each group?

Guiding question

How does species richness change through time for each group?

Summarizing the data

bird_spp_count <- birds_wy2025 |>

## Wrangling the data

### Pseudocode

Write pseudocode to create a data frame called `birds_wy2025` that:

- only includes the 2025 water year,
- excludes repeat observations, and 
- includes observations from the following eBird groups: Waterfowl; New World Sparrows; Herons, Ibis, and Allies; Shorebirds

**Pseudocode:**

- create object called `birds_wy2025` from `birds`
- filter date to keep observations after Sept 30 2024 and before Oct 1 2025 
- filter repeat_observations to only include "No" rows
- filter ebird_group to only include Waterfowl; New World Sparrows; Herons, Ibis, and Allies; Shorebirds
  
### Running code

We'll add a new function at the end of this code to insert the lowest taxonomic group for each eBird group name.  

We'll use `mutate()` (a function you already know) along with `case_when()`, which is a new function, to replace values in a column.

```{r}
#| label: cleaning and wrangling birds

# creating new object from birds
birds_wy2025 <- birds |> 
  # filter to include:
  # observation_dates in the 2025 water year
  # observations that are not repeats
  # eBird groups in the 4 groups of interest
  filter(observation_date > "2024-09-30" & observation_date < "2025-10-01",
         repeat_observation == "No",
         e_bird_group %in% c("Waterfowl",
                             "New World Sparrows",
                             "Herons, Ibis, and Allies",
                             "Shorebirds")) |> 
  # create a new column called lowest group
  mutate(lowest_group = case_when(
    # lowest group for sparrows is family
    e_bird_group == "New World Sparrows" ~ "Passerellidae",
    # lowest group for herons is family
    e_bird_group == "Herons, Ibis, and Allies" ~ "Ardeidae",
    # lowest group for shorebirds is order
    e_bird_group == "Shorebirds" ~ "Charadriiformes",
    # lowest group for waterfowl is family
    e_bird_group == "Waterfowl" ~ "Anatidae"
  ))
```

### Running code

Then, write the code that corresponds to your pseudocode.

```{r}
#| label: calculating total count of herons

# create new object from birds_wy2025
herons_total_count <- birds_wy2025 |> 
  # filter to only include wading birds
  filter(lowest_group == "Ardeidae") |>

  group_by(species) |> 
  # calculate total counts per species

  arrange(-total_count)
```

## Seasonal differences in species richness

### Guiding question

In what season is species richness highest for each eBird group?

:::{.callout-note title="What is species richness?"}

Species richness is the _number of species that occur in a given sampling unit_. In this case, we are interested in the number of species in a given group in a given season in the 2025 water year. Thus, we are calculating _species richness_ for four different seasons for four different eBird groups.

:::

### Pseudocode for summarizing

When used with `group_by()` and `summarize()`, `n_distinct()` allows you to count the number of distinct values in a group or multiple groups.  

We can use this combination of functions to determine the number of species observed in a given eBird group in a given season.  

Write pseudocode to calculate the number of species observed in each eBird group in a given season. 

**Pseudocode:**

- create new object from `birds_wy2025`
- "roll up" Belding's subspecies to be considered with Savannah Sparrows
- group by ebird group and season
- count number of species in each group in each season 
- ungroup dataframe

:::{.callout-note title="What is roll up?"}
Belding's is a subspecies of Savannah Sparrow. When doing any calculations of species richness, it wouldn't make sense to include both Savannah Sparrow and Belding's Savannah Sparrow.

Instead of counting subspecies as species, researchers tend to "roll up" subspecies; that is, count subspecies as the higher species. In this example, we'll use `mutate()` and `case_when()` to roll up the subspecies to make sure that Belding's (the subspecies) is counted as Savannah (the species).
:::

### Running code

```{r}
#| label: calculating species richness per group per season

# create new object from birds_wy2025
season_spp_count <- birds_wy2025 |>
  # roll up subspecies
  mutate(species = case_when(
    # whenever Belding's appears, replace with Savannah Sparrow
    species == "Savannah Sparrow (Belding's)" ~ "Savannah Sparrow",
    # keep all other values in species
    .default = species
  )) |> 
  # group data frame by eBird group and season
  group_by(e_bird_group, season) |> 
  # count unique species that appear in each season in each group
  summarize(num_spp = n_distinct(species)) |> 
  # ungroup data frame
  ungroup() |> 
  # relevel season to be in correct order for water year
  mutate(season = fct_relevel(season,
                              "Fall", "Winter", "Spring", "Summer"))
```

### Visualizing

```{r}
#| label: plotting species richness per group per season

# base layer: ggplot
ggplot(data = season_spp_count,
       # x-axis should be season, y-axis should be number of species
       mapping = aes(x = season,
                     y = num_spp,
                     fill = season)) +
  # first layer: columns for species richness
  geom_col() +
  # custom color for seasons
  scale_fill_manual(values = c(
    "Fall" = "#287984",
    "Winter" = "#9c8020",
    "Spring" = "#589101",
    "Summer" = "#4987c9"
  )) +
  # get rid of space between columns and x-axis
  scale_y_continuous(expand = c(0, 0),
                     limits = c(0, 12)) +
  # different theme from default
  theme_bw() +
  # facet figure to have one panel for each eBird group
  facet_wrap(~ e_bird_group) +
  # relabel x-and y-axes
  labs(x = "Season",
       y = "Species richness") +
  # take out the legend
  theme(legend.position = "none")
  
```

### Interpretation

When is species richness highest and lowest for each group?

Species richness for Herons is highest in Spring and Summer, but lowest in Fall and Winter (low throughout though).  

Species richness for New World Sparrows is highest in Winter and lowest in summer.  

Shorebird species richness is highest in winter and spring, but lowest in fall and summer.  

Waterfowl species richness is highest in winter and spring and lowest in fall and summer. 

## Seasonal differences in total counts

### Guiding question

What are differences in total counts between seasons?  

How does this differ (or not) from differences in species richness between seasons?

:::{.callout-note title="What are total counts?"}

In each survey, some number of individuals are counted for a species.  

For example, you might see a flock of 12 Song Sparrows at the beginning of your survey, and another flock of 5 Song Sparrows at the end.  

For that survey, your total count of Song Sparrows is 17. However, there is no way to know for sure if the 5 sparrows from the end of your survey are different from the 12 you saw at the beginning.

For this reason, we can only calculate "total counts": the total number of counted individuals in a given survey.

:::

### Running code to summarize data

First, write code to calculate the number of individuals counted in each eBird group in each season. 

```{r}
#| label: calculating total counts per group per season

# create new object from birds_wy2025
season_bird_count <- birds_wy2025 |>
  # group data frame by eBird group and season
  group_by(e_bird_group, season) |> 
  # calculate total counts for each group in each season

  # ungroup data frame
  ungroup() |> 
  # relevel seasons 
  mutate(season = fct_relevel(season,
                              "Fall", "Winter", "Spring", "Summer"))
```
