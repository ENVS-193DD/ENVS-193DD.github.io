Summary

Guiding questions

What are the temporal trends in Coyote-thistle (Eryngium vaseyi) percent cover across years?

What differences are there in Coyote-thistle percent cover across vernal pools?

Data science tools

To answer these questions, we are going to:

  • review reading in packages
    • library()
  • use familiar packages
    • tidyverse
    • here
  • introduce new packages
    • naniar to visualize missing data
    • patchwork to arrange multipanel figures
  • review reading in data
    • read_csv()
  • review cleaning and wrangling functions/operators
    • filter()
    • mutate()
    • left_join()
  • review summarizing functions
    • group_by(), summarize(), and ungroup()
  • review visualizations using ggplot(), including:
    • geom_point() and geom_line()
  • introduce a new visualization tool:
    • gg_miss_var() and gg_miss_upset() to visualize missing data
  • introduce new wrangling functions:
    • unite() from the tidyr package (within tidyverse) to create new columns with values separated by a character
    • str_detect() from the stringr package (within tidyverse) to detect any values in a character string

Implicit NAs

# create a new object from veg
pools_implicitNA <- veg |> 
  # filter all occurrences where transect_name contains VP
  filter(str_detect(transect_name, "VP")) |> 
  # group by year, pool, and species
  group_by(year, transect_name, psoc) |>
  # calculate mean percent cover
  summarize(mean_pc = mean(percent_cover, na.rm = TRUE)) |> 
  # ungroup data frame
  ungroup() |> 
  # create a new column called year_pool from year and transect name
  unite("year_pool", year, transect_name, remove = TRUE) |>
  # join with metadata by year_pool
  left_join(metadata, by = "year_pool")

Explicit 0s

# create a new object from veg
pools_explicit0 <- veg |> 
  # filter all occurrences where transect_name contains VP
  filter(str_detect(transect_name, "VP")) |> 
  # complete all possible combinations of year, pool, psoc and distance
  complete(year, transect_name, psoc, transect_distance,
           # fill values of percent cover with 0
           fill = list(percent_cover = 0)
           ) |>
  # group by year, pool, and psoc
  group_by(year, transect_name, psoc) |>
  # calculate total percent cover across all quads
  summarize(sum_pc = sum(percent_cover, na.rm = TRUE)) |>
  # ungroup the data frame
  ungroup() |>
  # create a new column called year_pool from year and transect name
  unite("year_pool", year, transect_name, remove = TRUE) |> 
  # join with metadata data frame
  left_join(metadata, by = "year_pool") |> 
  # calculate mean percent cover from total percent cover/number of quadrats
  mutate(mean_pc = sum_pc/num_quad)
